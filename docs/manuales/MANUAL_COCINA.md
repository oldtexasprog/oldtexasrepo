## Introducción

Este manual está diseñado para el personal de cocina de Old Texas BBQ. Como cocinero, tu pantalla principal es el **Tablero de Comandas**, donde verás todos los pedidos que necesitan preparación.

### Tus Responsabilidades

- Revisar los pedidos entrantes
- Marcar pedidos como "En Preparación"
- Marcar pedidos como "Listo" cuando estén terminados
- Atender pedidos urgentes prioritariamente
- Revisar personalizaciones y observaciones especiales

---

## Acceso al Sistema

### Iniciar Sesión

1. Abre el navegador en la tablet o computadora de cocina
2. Ve a la dirección del sistema
3. Ingresa tu correo y contraseña
4. Haz clic en **"Iniciar Sesión"**

> **Tip:** Puedes guardar la contraseña en el navegador para acceso rápido.

### Cerrar Sesión

Haz clic en tu nombre → **"Cerrar Sesión"**

## Tablero de Comandas

El tablero es tu herramienta principal. Muestra los pedidos en 3 columnas:

```
┌─────────────┬─────────────────┬─────────────┐
│  PENDIENTE  │ EN PREPARACIÓN  │    LISTO    │
├─────────────┼─────────────────┼─────────────┤
│  Pedido #1  │    Pedido #3    │  Pedido #5  │
│  Pedido #2  │    Pedido #4    │  Pedido #6  │
│     ...     │       ...       │     ...     │
└─────────────┴─────────────────┴─────────────┘
```

### Columnas del Tablero

| Columna | Descripción | Color |
|---------|-------------|-------|
| **Pendiente** | Pedidos recién llegados, esperando preparación | Amarillo |
| **En Preparación** | Pedidos que se están cocinando | Azul |
| **Listo** | Pedidos terminados, listos para entrega | Verde |

### Contador de Pedidos

Cada columna muestra un **badge** con la cantidad de pedidos:
- **Pendiente (5)** - 5 pedidos esperando
- **En Preparación (3)** - 3 pedidos cocinándose
- **Listo (2)** - 2 pedidos terminados

## Gestión de Pedidos

### Tarjeta de Comanda

Cada pedido se muestra como una tarjeta con:

| Elemento | Descripción |
|----------|-------------|
| **Número** | #1, #2, etc. (número del día) |
| **Canal** | Icono indicando origen (WhatsApp, Mostrador, etc.) |
| **Tiempo** | Minutos desde que llegó el pedido |
| **Productos** | Lista de items con cantidades |
| **Personalizaciones** | Modificaciones especiales (destacadas en color) |
| **Observaciones** | Notas del cliente o cajera |

### Leer una Comanda

```
┌──────────────────────────────────┐
│ #15 - WhatsApp        ⏱️ 5 min  │
├──────────────────────────────────┤
│ 2x Brisket Plate                 │
│    └ Extra salsa BBQ             │
│    └ Sin cebolla                 │
│ 1x Costillas (1 lb)              │
│ 3x Refresco cola                 │
├──────────────────────────────────┤
│ 📝 "Entregar a las 2pm"          │
├──────────────────────────────────┤
│ [ Iniciar Preparación ]          │
└──────────────────────────────────┘
```

### Iniciar Preparación

1. Identifica el pedido pendiente más antiguo (arriba)
2. Lee los productos y personalizaciones
3. Haz clic en **"Iniciar Preparación"**
4. La tarjeta se moverá a la columna "En Preparación"
5. Comienza a cocinar

### Marcar como Listo

1. Cuando termines de preparar el pedido
2. Haz clic en **"Marcar como Listo"**
3. La tarjeta se moverá a la columna "Listo"
4. El sistema notificará automáticamente a los repartidores

### Usar Drag & Drop (Arrastrar)

También puedes arrastrar las tarjetas entre columnas:
1. Haz clic y mantén presionado sobre la tarjeta
2. Arrastra hacia la columna deseada
3. Suelta para confirmar el cambio

## Alertas y Notificaciones

### Nuevo Pedido

Cuando llega un nuevo pedido:
- Escucharás un **sonido de alerta**
- La tarjeta aparecerá en "Pendiente"
- El contador de la columna aumentará

### Pedido Urgente

Un pedido se marca como **URGENTE** cuando:
- Lleva más de **30 minutos** sin completarse
- Se mostrará con **borde rojo**
- Aparecerá un **icono de alerta**

```
┌──────────────────────────────────┐
│ ⚠️ #8 - Urgente      ⏱️ 35 min  │
│ ──────────────────────────────── │
│ PRIORIZAR ESTE PEDIDO            │
└──────────────────────────────────┘
```

### Orden de Preparación

**Siempre prepara los pedidos en orden de llegada:**
1. Los más antiguos primero (arriba)
2. Urgentes tienen prioridad absoluta
3. Pedidos de mostrador pueden requerir atención inmediata

## Modo Pantalla Completa

Para una mejor visualización durante horas pico:

### Activar Pantalla Completa

1. Haz clic en el botón **"Pantalla Completa"** (esquina superior derecha)
2. El tablero ocupará toda la pantalla
3. Se ocultará el menú lateral

### Desactivar

- Presiona **Escape (Esc)** en el teclado
- O haz clic en **"Salir de Pantalla Completa"**

### Beneficios

- Más espacio para ver comandas
- Menos distracciones
- Mejor visibilidad desde lejos

## Preguntas Frecuentes

### No escucho las alertas de nuevos pedidos

1. Verifica que el volumen de la computadora/tablet esté alto
2. Asegúrate de que el navegador tiene permiso para reproducir audio
3. Haz clic en cualquier parte de la pantalla una vez (algunos navegadores bloquean audio hasta la primera interacción)

### Un pedido tiene información incorrecta

1. No prepares el pedido aún
2. Notifica a la cajera inmediatamente
3. Ella corregirá la información
4. Espera la actualización antes de continuar

### Se me pasó marcar un pedido como "En Preparación"

No te preocupes:
1. Búscalo en la columna "Pendiente"
2. Haz clic en "Iniciar Preparación"
3. El sistema actualizará el tiempo correctamente

### Necesito cancelar un pedido desde cocina

Los cocineros no pueden cancelar pedidos. Si hay un problema:
1. Contacta a la cajera
2. Ella cancelará el pedido si es necesario
3. Te aparecerá como "Cancelado" y desaparecerá del tablero

### El tablero no muestra pedidos nuevos

1. Verifica tu conexión a internet
2. Recarga la página (F5)
3. Si persiste, cierra sesión y vuelve a entrar
4. Contacta al administrador si continúa el problema

### ¿Qué hago si hay muchos pedidos urgentes?

1. Prioriza por orden de llegada (el más antiguo primero)
2. Si tienes ayuda, divide los pedidos
3. Mantén la calma y trabaja de manera sistemática
4. Notifica al encargado si la situación es crítica

## Buenas Prácticas

### Durante el Servicio

1. **Revisa el tablero constantemente** - Cada 2-3 minutos mínimo
2. **Lee las personalizaciones** - Antes de comenzar a preparar
3. **Marca "En Preparación"** - Tan pronto inicies un pedido
4. **Marca "Listo"** - Inmediatamente al terminar
5. **No acumules pedidos listos** - Deben salir rápido

### Comunicación

1. Si algo no está claro, **pregunta a la cajera**
2. Si falta un ingrediente, **notifica inmediatamente**
3. Si hay retraso, **informa al encargado**

### Al Inicio del Turno

1. Verifica que la pantalla funcione
2. Prueba el sonido de alertas
3. Revisa si hay pedidos pendientes de turnos anteriores

### Al Final del Turno

1. Verifica que no queden pedidos en preparación
2. Los pedidos "Listo" deben ser recogidos
3. Cierra sesión en el sistema
