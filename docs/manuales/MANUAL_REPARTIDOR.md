## Introducción

Este manual está diseñado para los repartidores de Old Texas BBQ. Como repartidor, tu aplicación te permite ver pedidos disponibles, aceptar entregas, confirmar entregas y liquidar con caja.

### Tus Responsabilidades

- Revisar pedidos listos para recoger
- Aceptar y entregar pedidos asignados
- Confirmar entregas en el sistema
- Cobrar y entregar cambio correctamente
- Liquidar con caja al final del día
- Reportar cualquier incidencia

---

## Acceso al Sistema

### Iniciar Sesión

1. Abre el navegador en tu teléfono
2. Ve a la dirección del sistema
3. Ingresa tu correo y contraseña
4. Haz clic en **"Iniciar Sesión"**

> **Tip:** Agrega la página a tu pantalla de inicio para acceso rápido como si fuera una app.

### Agregar a Pantalla de Inicio (Android)

1. Abre el sitio en Chrome
2. Toca los tres puntos (⋮) en la esquina
3. Selecciona **"Agregar a pantalla de inicio"**
4. Confirma el nombre y toca **"Agregar"**

### Agregar a Pantalla de Inicio (iPhone)

1. Abre el sitio en Safari
2. Toca el botón de compartir (□↑)
3. Selecciona **"Agregar a inicio"**
4. Confirma tocando **"Agregar"**

## Aceptar Pedidos

### Ver Pedidos Disponibles

1. En la sección **"Pedidos Listos para Recoger"** verás los pedidos disponibles
2. Cada tarjeta muestra:
   - Número de pedido
   - Colonia de entrega
   - Monto total
   - Costo de envío

### Información del Pedido

Antes de aceptar, puedes ver:

| Dato | Descripción |
|------|-------------|
| **Colonia** | Zona de entrega |
| **Monto** | Total a cobrar |
| **Envío** | Costo del envío (tu comisión) |
| **Método de Pago** | Efectivo, Tarjeta, Transferencia |
| **Observaciones** | Instrucciones especiales |

> **Nota:** Por seguridad, no se muestra el teléfono completo ni el nombre completo del cliente hasta que aceptes el pedido.

### Aceptar un Pedido

1. Revisa la información del pedido
2. Haz clic en **"Aceptar Pedido"**
3. El pedido pasará a "Mis Pedidos Asignados"
4. El estado cambiará a **"En Reparto"**
5. Ve a cocina a recoger el pedido

### ¿Qué pasa al aceptar?

- La cajera y cocina ven que tú tienes el pedido
- El pedido desaparece de la lista de otros repartidores
- Se registra la hora de asignación

## Realizar Entregas

### Ver Detalles del Pedido

En "Mis Pedidos Asignados", toca **"Ver Detalles"** para ver:

| Información | Descripción |
|-------------|-------------|
| **Cliente** | Nombre completo |
| **Dirección** | Dirección de entrega |
| **Teléfono** | Para contactar si es necesario |
| **Productos** | Lista de items del pedido |
| **Observaciones** | Instrucciones especiales |
| **Método de Pago** | Cómo va a pagar |
| **Total** | Monto a cobrar |
| **Cambio** | Dinero a devolver (si aplica) |

### Realizar la Entrega

1. Llega a la dirección indicada
2. Entrega el pedido al cliente
3. Cobra según el método de pago:
   - **Efectivo:** Cobra el total, da el cambio indicado
   - **Tarjeta:** Ya está pagado
   - **Transferencia:** Ya está pagado
4. Confirma la entrega en el sistema

### Confirmar Entrega

1. Haz clic en **"Marcar como Entregado"**
2. Confirma la acción
3. El pedido pasará a "Liquidaciones Pendientes"

> **Importante:** Solo marca como entregado cuando realmente hayas entregado el pedido.

## Liquidaciones

### ¿Qué es una Liquidación?

Es el proceso de entregar a caja el dinero de los pedidos en efectivo.

### Pedidos Pendientes de Liquidar

En la sección **"Liquidaciones Pendientes"** verás:
- Pedidos entregados con pago en efectivo
- Monto total a entregar
- Tu comisión (envío)

### Cálculo de Liquidación

```
Monto a Entregar = Total del Pedido - Tu Comisión (Envío)
```

**Ejemplo:**
- Total del pedido: $350
- Costo de envío (tu comisión): $50
- Entregas a caja: $300

### Realizar Liquidación

1. Ve a caja con el dinero de tus entregas
2. La cajera verificará los montos
3. Entrega el efectivo correspondiente
4. La cajera marcará los pedidos como "Liquidados"
5. Verás los pedidos en tu "Historial del Día"

### Resumen de Liquidación

El sistema te muestra:

| Concepto | Ejemplo |
|----------|---------|
| **Pedidos a liquidar** | 5 pedidos |
| **Total cobrado** | $1,750 |
| **Tu comisión** | $250 |
| **A entregar** | $1,500 |

## Reportar Incidencias

### ¿Qué es una Incidencia?

Cualquier problema que ocurra durante la entrega:
- Cliente no contesta
- Dirección incorrecta
- Cliente rechaza el pedido
- Producto dañado
- Otro problema

### Cómo Reportar

1. En el pedido asignado, haz clic en **"Reportar Incidencia"**
2. Selecciona el tipo de incidencia
3. Describe el problema brevemente
4. Envía el reporte

### Tipos de Incidencias

| Tipo | Descripción |
|------|-------------|
| **Cliente no contesta** | No responde llamadas ni mensajes |
| **Dirección incorrecta** | La dirección no existe o es errónea |
| **Cliente rechazó** | El cliente no quiere el pedido |
| **Producto dañado** | El pedido se dañó durante el reparto |
| **Otro** | Cualquier otro problema |

### ¿Qué pasa después?

1. El encargado recibe una notificación
2. La cajera es informada
3. Ellos te indicarán cómo proceder
4. El pedido queda registrado con la incidencia

## Preguntas Frecuentes

### El cliente me pagó con tarjeta pero el sistema dice "Efectivo"

1. No aceptes el pago con tarjeta si el sistema indica efectivo
2. El cliente debe pagar como indica el pedido
3. Si hay confusión, contacta a la cajera

### El cliente no tiene cambio exacto

1. Si el cliente da más dinero del esperado, da el cambio correcto
2. Registra mentalmente la diferencia
3. Al liquidar, explica a la cajera la situación

### No encuentro la dirección

1. Intenta llamar al cliente (el teléfono está en los detalles)
2. Si no contesta, espera 5 minutos y vuelve a intentar
3. Si persiste, reporta incidencia "Dirección incorrecta"
4. Regresa con el pedido y notifica a caja

### El pedido está incompleto o dañado

1. No entregues el pedido dañado
2. Reporta incidencia inmediatamente
3. Regresa con el pedido a la sucursal
4. La cajera decidirá los siguientes pasos

### ¿Puedo aceptar varios pedidos a la vez?

Sí, puedes tener varios pedidos asignados si van a zonas cercanas. Pero:
- Asegúrate de poder entregarlos a tiempo
- Los pedidos urgentes tienen prioridad
- No aceptes más de lo que puedes manejar

### Olvidé marcar un pedido como entregado

1. Márcalo lo antes posible
2. El sistema registra la hora actual
3. Si hay diferencia significativa, notifica a caja

### No me aparecen pedidos disponibles

1. Verifica tu conexión a internet
2. Recarga la página
3. Puede que no haya pedidos listos en este momento
4. Si persiste, contacta a caja

## Consejos para Repartidores

### Antes de Salir

1. Verifica que tienes suficiente cambio
2. Revisa la dirección y observaciones
3. Confirma el monto a cobrar
4. Asegúrate de que el pedido esté completo

### Durante la Entrega

1. Trata al cliente con respeto
2. Confirma el nombre antes de entregar
3. Cobra primero, luego entrega
4. Da el cambio correcto

### Después de Entregar

1. Marca como entregado inmediatamente
2. Revisa si hay más pedidos disponibles
3. Liquida con caja antes de terminar tu turno

### Seguridad

1. No muestres grandes cantidades de efectivo
2. Liquida frecuentemente si tienes mucho dinero
3. Reporta cualquier situación sospechosa
4. Mantén tu teléfono cargado
